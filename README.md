# 📦 Sistema de Controle de Estoque - Console App

Este é o meu primeiro sistema completo de controle e gerenciamento de estoque para peças automotivas, desenvolvido em **C#** utilizando uma aplicação do tipo Console (Terminal). Este projeto foi construído para solidificar conceitos fundamentais de lógica de programação e orientação a objetos.

## 🛠️ Tecnologias e Conceitos Utilizados
*   **Linguagem:** C# (.NET)
*   **Paradigma:** Orientação a Objetos (Classes, Atributos, Construtores e Métodos)
*   **Estruturas de Dados:** Listas genéricas (`List<T>`) para manipulação de dados em memória
*   **Lógica Principal:** Laços de repetição (`while`), estruturas condicionais (`if/else`) e tratamento de menus interativos

## 🚀 Funcionalidades do Sistema
*   **Cadastrar Peças:** Permite a inserção de novos produtos com Nome, Quantidade e Preço.
*   **Listar Estoque:** Exibe em formato de tabela todas as peças armazenadas em memória.
*   **Atualizar Quantidade:** Permite realizar a entrada ou saída de itens do estoque de forma dinâmica.
*   **Remover Itens:** Exclusão de peças do gerenciamento.
